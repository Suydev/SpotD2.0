🚀 Run Anywhere

SpotDL is a single-file Flask app — install in seconds on a laptop, server, or even Android (Termux).

⚡ One-Line Install (Linux / macOS / Termux)

Clones the repo, installs Python deps + ffmpeg, starts the server on http://localhost:5000.

curl -fsSL https://raw.githubusercontent.com/yourname/spotdl/main/install.sh | bash

Or clone & run manually:

git clone https://github.com/yourname/spotdl && cd spotdl bash install.sh # installs deps + starts server
📱 Android (Termux)
  1. Install Termux from F-Droid
  2. Run: pkg update && pkg install python ffmpeg git -y
  3. Run: git clone https://github.com/yourname/spotdl && cd spotdl
  4. Run: pip install -r requirements.txt
  5. Run: python src/web_app.py & open http://localhost:5000
🖥 Server / VPS (production)

Run with gunicorn behind a reverse proxy:

pip install -r requirements.txt gunicorn --bind 0.0.0.0:5000 --workers 2 --timeout 600 --chdir src web_app:app
🐳 Docker
docker compose up -d # starts on port 5000

Volumes spotd_data and spotd_downloads persist your settings and downloaded files between restarts.

🔧 Environment Variables (all optional)
PORT Port to listen on (default 5000)
SESSION_SECRET Random string for Flask sessions (auto-generated if missing)
FLASK_DEBUG Set to true for dev mode

✅ No Spotify or YouTube credentials needed. All metadata comes from public embed pages.

ℹ️ Notes
  • ffmpeg is required for audio conversion (installed automatically by install.sh)
  • Files persist on the server until they exceed the retention window (configure in Settings)
  • You can re-download the same ZIP as many times as you want from the Downloads page
  • Respect YouTube and Spotify terms of service when sharing this app
