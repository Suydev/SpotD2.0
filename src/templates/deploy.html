🚀 Deploy Guide

Run SpotDL on your own server for free

🐳 Docker (Recommended)

Spin up the app anywhere Docker runs — your VPS, Raspberry Pi, or any cloud VM.

git clone https://github.com/yourname/spotdl cd spotdl cp .env.example .env # add your Spotify creds docker compose up -d # starts on port 5000

Requires Docker & Docker Compose. The data/ folder is mounted as a volume to persist credentials.

☁️ Koyeb (Free, no credit card)
  1. Sign up at koyeb.com (free tier available)
  2. Create a new App → choose Docker source
  3. Point to your GitHub repo or paste the Docker image
  4. Add env vars: SPOTIFY_CLIENT_ID and SPOTIFY_CLIENT_SECRET
  5. Set port to 5000 and deploy
🔑 Environment Variables
SPOTIFY_CLIENT_ID Your Spotify app client ID (required)
SPOTIFY_CLIENT_SECRET Your Spotify app client secret (required)
SESSION_SECRET Random string for Flask sessions (optional)
PORT Port to listen on (default: 5000)
ℹ️ Notes
  • SpotDL uses yt-dlp under the hood — no Spotify Premium required
  • The Spotify API is used only to look up track metadata (100% free tier)
  • Downloaded files are deleted from the server 60 seconds after you save them
  • FFmpeg must be installed in the environment for audio conversion (included in Docker image)
  • Do not expose this app publicly on a large scale — respect YouTube & Spotify ToS
