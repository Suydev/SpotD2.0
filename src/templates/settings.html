ℹ️ How Downloads Work
  • Spotify is used only to read track names (title + artist) from public playlists
  • Each track is then found and downloaded from YouTube via yt-dlp — completely free
  • Audio is converted to your chosen quality using FFmpeg
  • All tracks in a batch are bundled into a ZIP file for easy download
  • Files are deleted from the server 60 seconds after you download them
  • No accounts, no logins, no subscriptions — ever
